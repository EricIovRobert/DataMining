# Data Mining Project – Working Instructions

## Project
Heart Disease Prediction (Project 4)
- Dataset: UCI Heart Disease (Cleveland)
- Task: Feature selection + classification
- Methods: PCA, Naive Bayes, Decision Trees

## How We Work
- Incremental: one step at a time
- First write the Python notebook code for a step
- Then write the report section corresponding to that step
- Only move to the next step after the current one is done

## Code Rules
- Copy-paste ready blocks (user uploads to Colab manually)
- User runs the code and shares results back
- KISS: keep it as simple as possible
- Minimal comments — only what's necessary to understand

## Step Order
1. Data loading & exploration → report: Description of the dataset
2. Data preprocessing → report: Processing flow (part 1)
3. Feature selection with PCA → report: Processing flow (part 2)
4. Classification: Naive Bayes → report: Results (part 1)
5. Classification: Decision Trees → report: Results (part 2)
6. Comparison → report: Results (comparison)
7. Conclusions → report only
